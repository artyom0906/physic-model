# openGlEngineStudent


This application was created to simulate the behavior of gas particles

## Run
   
* Linux
    
    You need java 1.8 version
    ```bash
    java -jar modelling.jar
    ```
  
* Windows

    You need java 1.8 version
    ```
    java -jar modelling.jar
    ``` 
    or double-click on file
